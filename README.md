# gorandom

## A go package for random things

# Usage

## Functions

### randStr()

``gorandom.randStr(number of characters)``

### randInt()

``gorandom.randInt(number of characters)``

## Example Program

```go
package main

import (
  "github.com/TheBaconPug/gorandom"
)

func main() {
  println(gorandom.randStr(10))
}
```
